viewvclog2rcs

ViewVC's Commit Log HTML To RCS File Converter

SYNOPSIS

  perl bin/viewvclog2rcs.pl --log-uri http://example/viewvc/path/to/file

DESCRIPTION

The viewvclog2rcs script reconstructs an RCS file from ViewVC (or ViewCVS)'s commit log HTML page of a CVS version controlled file.

OPTIONS

--log-uri=url

The URL of the commig log HTML page of ViewVC (or ViewCVS).

SEE ALSO

ViewVC <http://viewvc.org/>.
